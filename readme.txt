键位一览：
鼠标点击   触控
↑↓←→     方向键
shift    start键（跳过或者自动对话）
x  ⚪键
z   x键
c  ▢键
v  △键
q  L键
e  R键
ctrl 快速强制跳过
alt+enter 全屏幕显示（退出全屏也是这个）
esc 结束游戏（会有弹窗确认）
（以上键位仅供参考，小键盘亦有对应键位，有需要可自行探索）
除必须触控以外的操作均可以使用手柄代替，一般手柄基本都能支持。（经验证xbox手柄也可用）
游戏中可以随时呼出菜单进行存档/快速存档或者进行读取，对话回想并跳转到本场景之前的指定位置。
游戏设置也可以在游戏前和游戏中随时调节，如文字显示速度。
由于游戏存在闪退现象（尤其是在长时间无操作时），请务必及时手动存档。
在正式开始游玩之前，为了避免不必要的损失，必须首先验证能否存档/读档！
游戏存档文件在系统盘的这个位置,存档可以通过复制文件来备份，也可以分享给其他的玩家使用！：
C:\Users\（你的系统用户名）\Documents\My Games\mages_dmm\USAGI
